# earthquake-early-warning-receiver
## 概要
翻訳してみたら分かる通り、「緊急地震速報受信機」です。
即時に緊急地震速報(警報)を受信して表示するものです。
## API詳細
<a href="https://2p2quake.net>P2P地震情報</a>
を使用しました。
URLは
<a href="https://api.p2pquake.net/v2/history?codes=556&limit=1&offset=0>こちら</a>
です。
